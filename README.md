# Mini SaaS Study

Monorepo com frontend em Next.js e backend em NestJS.

## Objetivo

Projeto de estudo e portfólio para demonstrar arquitetura full stack moderna em monorepo, com tipagem ponta a ponta e organização escalável.

## Visao geral

- Gerenciador: `pnpm` (workspaces)
- Orquestracao: `turbo`
- Frontend: `apps/web` (Next.js, porta `3001`)
- Backend: `apps/backend` (NestJS, porta `3000`)
